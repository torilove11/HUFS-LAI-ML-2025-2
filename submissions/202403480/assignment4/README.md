# Assignment 4: Data Collection & Analysis
**데이터 출처**: 디스코드 봇으로 직접 수집한 원본 영수증 이미지
- **제출 파일**
  - `data/receipts.csv` (수집 결과: 카테고리, 금액, 일시 등)
  - `data/vendor_map.csv` (업체명/키워드 → 카테고리 매핑표)
  - `data-analysis.ipynb` (간단 EDA/시각화)
- **데이터 개요**
  - 샘플 수: 15개 (실제 사용한 영수증만으로 수집을 하다보니 데이터 표본이 너무 적게 수집이 된 것 같아, 좋은 모델 학습을 위해 더 많은 영수증을 모을 수 있도록 할 예정입니다!)
- **분석 목표**
  - 영수증에서 **카테고리/금액/일시** 자동 추정 → 수동 수정으로 정답 확정 
  - 분포/결측 확인 등 기초 EDA 수행
- **결론 요약**
  - 디스코드 봇에서 OCR이 거의 제대로 진행이 되지 않아 대부분 수동 수정을 통해 정답을 확정했어서, OCR에 관련해 더 찾아봐야 할 것 같다고 느꼈습니다.
  - 업체명 사전(`vendor_map.csv`) 확장을 통해 카테고리 자동 추정 정확도 개선을 해보려 하였지만, 이 부분 또한 미흡한 모습을 보여 다시 한 번 고민해봐야 할 것 같다고 생각했습니다.
  - 다음 단계: 필요시 이미지 전처리(이진화/샤프닝/왜곡보정) 및 한글 OCR 튜닝, 금액/일시 정규표현식 고도화
