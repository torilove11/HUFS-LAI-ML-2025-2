# 5th Assignment: Professor's Peace of Mind Sender

## 프로젝트 소개
한국외대 '기계학습의 이해' 과제로 수행된 프로젝트입니다.
메일이나 메시지가 교수님께 보내기에 적절한 '격식' 표현인지, 친구에게 쓰는 '비격식' 표현인지 판별합니다.

## 모델 성능
- Accuracy: 약 90% 이상 (Training 결과 참조)
- 사용 모델: 3-Layer Neural Network (Batch Normalization 적용)
