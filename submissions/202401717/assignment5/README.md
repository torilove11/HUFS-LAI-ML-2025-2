# 모델 아키텍처
- 다항 회귀를 사용함.
  - 점수를 예측하는 task 이고, feature의 수가 많지 않으므로 과적합을 방지하기 위해 Dense layer를 사용하지 않음.
- 여러 개의 feature를 통해 target을 회귀 예상하는 모델
# 평가 지표 (eval)
- 회귀모델이므로 MSE, MAE, R2 사용
- MSE(평균제곱오차): 6.465124
  - 편차의 제곱 평균으로, 점수의 제곱값이 평균적으로 6점 정도 차이난다는 뜻이다.
  - 개선의 여지가 있으나, 대략적인 점수 예측을 위한 모델로 활용 가능하다.
- MAE(평균절대오차): 1.434646
  - 편차의 절댓값 평균으로, 다소 낮은 수준이다.
- R2(R-squared, 결정계수): 0.586576
  - 0과 1 사이의 값으로, 1에 가까울수록 독립변수들이 종속변수 값을 잘 설명한다.
  - 0.5 이상의 값이 도출된 것을 보면 낮지 않다.
# 성능 결과
- inference 결과, 지표에 큰 차이가 없음.
- 실제 측정 데이터에 대해서도 유사한 성능을 기대할 수 있음.
- 모델 성능 개선의 여지가 있으나, 대략적인 예측을 위해 활용 가능함.
