# 분석 결과 요약

## 데이터 개요
- 수집된 뉴스 데이터는 총 1800개입니다.
- 데이터 컬럼:
  - `keyword`: 검색에 사용된 키워드 (예: IT, 경제, 정치, AI, 사회, 국제)
  - `title`: 뉴스 제목
  - `description`: 뉴스 요약문
  - `link`: 뉴스 원문 URL
  - `pubDate`: 뉴스 발행일시

## 기초 통계
- 제목 길이(`title_length`): 대부분 20~45자에 분포
- 요약문 길이(`desc_length`): 대부분 125~150자에 분포

## 시각화 결과
1. **키워드별 뉴스 개수**
   - 모든 키워드에서 뉴스 개수는 대략 300개로 균일하게 분포
2. **제목 길이 분포**
   - 20~45자 구간에 뉴스가 집중
3. **요약문(description) 길이 분포**
   - 125~150자 구간에 뉴스가 집중

> 전반적으로 뉴스 데이터는 키워드별 균일하게 수집되었으며, 제목과 요약문의 길이가 일정한 범위에 집중되어 있음이 확인되었습니다.
