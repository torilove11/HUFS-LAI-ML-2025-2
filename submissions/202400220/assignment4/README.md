## 주요 변경 사항

- 기존 연구 제안서대로 영->한 기능도 포함하려 했으나, 오히려 영문을 한국어로 번역 시 가독성이 떨어지는 경우가 많이 발생하였습니다.
- 따라서 해당 기능은 실 사용 시 큰 도움이 되지 못하는 경우가 많을 것이라 판단하여 한->영 데이터만 수집하였습니다.

## 분석 결과

**1) 데이터 요약**
- 제 의도, 기계 번역, 제 번역의 세 colum을 가진 총 100쌍의 한 -> 영 데이터 행렬을 제작하였습니다.
- 데이터는 바로 복사해서 사용하는 환경과 동일하도록 줄바꿈을 포함하였습니다.
- 거의 모든 기계 번역은 제 번역과 차이가 존재하였습니다.
- 결측값은 존재하지 않았습니다.

**2) 통계적 특징**
- **문장 길이:** 기계 번역의 평균 길이는 약 11.6단어인 반면, 제 번역은 약 10.7단어로 제가 좀 더 간결하게 번역하는 경향이 있습니다.
- **분포:** 한국어로 이루어진 제 의도는 3단어에서 16단어까지 다양하게 분포되어 있어, 짧은 대사부터 긴 지문까지 전부 포함하고 있습니다.

**3) 산점도 분석**
- `Machine vs. My Translation Length (Words)` 그래프에서 더 많은 점들이 빨간 점선(y=x) 아래쪽에 위치합니다.
- 이는 기계 번역보다 제 번역이 단어 수가 적은 경우가 많다는 것을 시각적으로 보여줍니다.
- 그리고 이는 기계 번역이 지나치게 장황하거나 단어의 생략을 잘 사용하지 않음을 시사합니다.
